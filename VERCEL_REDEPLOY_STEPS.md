# Vercel Redeploy Steps (With Screenshots Guide)

## Step 1: Click on Frontend Project
- Click on **"hospital-management-99ih"** project
- (URL: hospital-management-99ih.vercel.app)

## Step 2: Go to Deployments Tab
- Top menu mein **"Deployments"** tab par click karo

## Step 3: Find Latest Deployment
- Sabse upar wali deployment (most recent) ko dekho
- Right side mein **three dots (⋯)** button dikhega

## Step 4: Click Three Dots Menu
- Three dots (⋯) par click karo
- Dropdown menu khulega

## Step 5: Select Redeploy
- Menu mein **"Redeploy"** option par click karo
- Ek dialog box khulega

## Step 6: Uncheck Build Cache (Important!)
- Dialog box mein **"Use existing Build Cache"** checkbox dikhega
- Is checkbox ko **UNCHECK** karo (taaki fresh build ho)

## Step 7: Confirm Redeploy
- **"Redeploy"** button par click karo
- Deployment start ho jayegi

## Step 8: Wait for Completion
- 2-3 minutes wait karo
- Status "Building" → "Ready" ho jayega

## Step 9: Test Application
- Browser mein jaao: https://hospital-management-99ih.vercel.app
- Hard refresh karo: **Ctrl + Shift + R** (Windows) or **Cmd + Shift + R** (Mac)
- Register/Login test karo
